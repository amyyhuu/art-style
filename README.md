# Art Style

A Codex skill for cute, pink, polished art direction, prompts, critique, and visual style guidance.

## Use This Skill

Use `$art` when you want Codex to make an idea feel cute, pink, aesthetic, cozy, polished, or ready for image generation.

Example prompts:

- `Use $art to make this sticker idea cuter.`
- `Use $art to write a prompt for a Strawberry Milk Pet icon.`
- `Use $art to critique this illustration.`
- `Use $art to make concept art directions for a soft bunny cafe mascot.`

The skill includes a named **Strawberry Milk Pet** style reference with palette, shape language, composition recipes, prompt formula, and quality checks.
